# Instituto SOPA — site estático para GitHub Pages

Projeto acadêmico de reconstrução/modernização em HTML, CSS e JS puro.

## Como publicar
1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta.
3. Vá em **Settings > Pages**.
4. Em **Source**, escolha **Deploy from a branch**.
5. Branch: `main`; pasta: `/root`.
6. Salve e aguarde o link `https://seu-usuario.github.io/nome-do-repositorio/`.

## Ajustes recomendados antes de usar de verdade
- Trocar imagens de banco por fotos autorizadas da ONG.
- Confirmar números atuais de impacto, endereço, WhatsApp e chave Pix oficial.
- Conectar formulários a Google Forms, Formspree ou Netlify Forms.
- Inserir PDFs reais de prestação de contas.
